# Problem 0016: Sorting Basics

## Problem Summary
Master fundamental sorting algorithms and techniques.

## Key Concepts
- **Comparison-based**: Compare elements for ordering
- **Stable Sorting**: Maintain relative order of equal elements
- **In-place**: Sort without extra space
- **Divide & Conquer**: Break problem into subproblems

## Solution Approaches
1. **Bubble Sort**: Simple, O(n^2)
2. **Quick Sort**: Divide & conquer, O(n log n) average
3. **Merge Sort**: Stable, O(n log n)
4. **Heap Sort**: In-place, O(n log n)
5. **Built-in**: Use language sorting

## Time-Space Complexity
- **Quick Sort**: O(n log n) average, O(n^2) worst
- **Merge Sort**: O(n log n) always, O(n) space
- **Heap Sort**: O(n log n) always, O(1) space

## Interview Tips
- Compare algorithm trade-offs
- Discuss when to use each algorithm
- Stability and space considerations
