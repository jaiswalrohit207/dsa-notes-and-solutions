# Problem 0013: Linked List Basics

## Problem Summary
Master fundamental linked list operations.

## Key Concepts
- **Node Structure**: Value and next pointer
- **Traversal**: Moving through nodes
- **Operations**: Insert, delete, reverse

## Solution Approaches
1. **Iterative**: Loop through list
2. **Recursive**: Recursive traversal

## Time-Space Complexity
- **Iterative**: O(n) time, O(1) space
- **Recursive**: O(n) time, O(n) space for call stack

## Interview Tips
- Explain node structure clearly
- Handle null pointers carefully
- Show various traversal patterns
