# Problem 0015: Hash Map Basics

## Problem Summary
Master hash map and hash table operations.

## Key Concepts
- **Key-Value Pairs**: Store and retrieve by key
- **Hash Function**: Map keys to indices
- **Collision Resolution**: Handle multiple keys at same index
- **Performance**: O(1) average case operations

## Solution Approaches
1. **Separate Chaining**: Linked lists at each index
2. **Open Addressing**: Find alternative index
3. **Built-in Maps**: Use language collections

## Time-Space Complexity
- **Average Case**: O(1) insertion, deletion, lookup
- **Worst Case**: O(n) when many collisions

## Interview Tips
- Discuss hash functions
- Collision handling strategies
- Load factor and resizing
