# Problem 0014: Stack and Queue Basics

## Problem Summary
Master stack and queue data structures and operations.

## Key Concepts
- **Stack (LIFO)**: Last In, First Out
- **Queue (FIFO)**: First In, First Out
- **Push/Pop**: Stack operations
- **Enqueue/Dequeue**: Queue operations

## Solution Approaches
1. **Array-based**: Use array with pointer
2. **Linked List-based**: Use linked list
3. **Built-in Collections**: Use provided classes

## Time-Space Complexity
- **Push/Pop**: O(1) time each
- **Enqueue/Dequeue**: O(1) time each

## Interview Tips
- Clarify use cases for each
- Discuss implementations
- Handle edge cases
