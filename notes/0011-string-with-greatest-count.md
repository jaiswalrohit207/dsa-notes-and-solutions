# Problem 0011: String with Greatest Count of Characters

## Problem Summary
Find which string has the greatest number of characters.

## Key Concepts
- **String Comparison**: Compare string lengths
- **Iteration**: Loop through all strings
- **Maximum Tracking**: Track maximum length seen

## Solution Approaches
1. **Linear Search**: Iterate and track max
2. **Built-in Functions**: Use max() with key parameter

## Time-Space Complexity
- **Linear Search**: O(n) time, O(1) space

## Interview Tips
- Simple problem for warmup
- Discuss edge cases: empty array, tied lengths
- Mention Python max() approach
